Notice (for the content field):
• The recommended aspect ratios are 16x9
• Create a superscript (appears half a character above the normal line) by <sup>This is superscript</sup>
• Create a link that points to other place in the same article: First, create a link by <a name="mylink"></a>. Second. jump to it by <a href="#mylink">[Jump to link]</a>
• Create a superscript that links to other content in the same article: First, create a link by <a name="mytitle"></a> (put it in front of the title). Second, create a superscript by <sup><a href="#mytitle">[Jump to title]</a></sup>
• Images will not be autosaved, and hence the link you get after uploading images will be invalid once you have left/reload this page. So make sure you re-upload images if continue editing.
